This plugin logs the visitors of your homepage.

== Description ==
This plugin logs the visitors of your homepage. The records including IP-address, browser, operating system, requested page as well as the originate page. The extended version of this plugin also measures the actual time a visitor looks at the page.

== Installation ==
To install, simply click on «Install Now» and then on «Activate Plugin».

== Screenshots ==
1. The «Page Views» tab shows the visitors including IP-address, browser, operating system, requested page as well as the originate page.
2. The «Time of View» tab shows how long a visitor looks at a page. This feature is only available in the extended version.
